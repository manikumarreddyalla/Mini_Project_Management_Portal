# Mini Project Management Portal

A full-stack task management application built with React, Node.js/Express, and MySQL.

## Overview
This is my 4th year CSE major project where I'm building a complete task management application. Users can:
- View all tasks
- Create new tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks by status

## Technologies Used
- **Frontend:** React.js, Axios, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MySQL
